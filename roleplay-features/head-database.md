# Head Database

##

## Commands

| Commands                                  | Description                               |
| ----------------------------------------- | ----------------------------------------- |
| `/headdb` or `/hdb` or `/heads`           | Opens the main user interface.            |
| `/hdb info` or `/hdb i`                   | Shows info about the plugin, and database |
| `/hdb search [input]` or `/hdb s [input]` | Search for heads in the database.         |

Each head costs $100.
